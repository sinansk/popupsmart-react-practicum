# React Crud Todo List

Popupsmart Practicium First Case  
It is a crud todo app with [Mock Api](https://mockapi.io/).

## Run project on local  

##### You need to run the commands in project directory. 

### `npm i`

### `npm start`  

## Technologies

- `React`
- `Redux Toolkit`
- `Tailwind`
- `React Hot Toast`
- `Axios`
- `React Router`
